# 盒子模型

## 組成

- 內容區域 - width & height
- 內邊距 - padding (出現在內容與盒子邊緣)
- 邊框線 - border
- 外邊距 - margin (盒子外面) 隔開各個盒子的距離

## 1.border(bd)-邊框線

屬性值:邊框線粗細 線條樣式 顏色 (不分順序)

solod(實線) dashed(虛線) dotted(點線)

### border+方位名詞(bd+方位首字母ex.bdl)

屬性值:邊框線粗細 線條樣式 顏色

border-top/right/left/bottom

ex:border-bottom: 1px solid #5EB69C;

Hover配合border使用時 模型會被border的寬度往上位移 所以可以使用透明度來設定hover 讓hover控制border的顯示 

## 2.padding/padding-方位名詞(pd/pd+方位名詞)-內邊距

padding: 30px

padding-top/right/left/bottom 30px

### padding複合屬性

一個值 padding:10px (四方向10px)

四個值 padding: 10px 20px 30px 40px (順序上右下左)

三個職 padding: 10px 20px 30px (順序 上 左右 下)

兩個值 paddibg:10px 80px (順序 上下 左右)

## 3.margin(md/md+方位名詞)-外邊距

外邊距不會撐大盒子尺寸

### 盒子模型-尺寸計算

盒子尺寸=內容尺順+border尺寸+padding尺寸

給盒子+bd+pd會撐大盒子

- 手動作減法,減去border/padding尺寸後,重新計盒子尺寸
- 內減模式: box-sizing:border-box

## 版心居中 : margin: 10px auto 10px (盒子需要有寬度)

當中的auto 是以父級為準 計算邊框的距離 來將元素置中

## 清除預設樣式

清除標籤預設的樣式 例如:h1標籤中的內外邊距

用通配選擇器 或並級選擇器 選中所有標籤

**標籤內可補上內減模式: box-sizing:border-box**

**清除列表符號標籤 li{ list-style:none**

**超連接預設顏色清除 a {text-decoration**

## 盒子模型-元素溢出

屬姓名:overflow

屬性值:

hidden 溢出隱藏

scroll 溢出滾動(無論有沒有溢出都顯示滾動條)

auto 溢出滾動(溢出才顯示滾動條位置)

## 盒子模型-圓角

屬姓名:border-radius

屬性值:

數字+px/百分比

## 盒子模型-陰影

屬性名:box-shadow

屬性值: x軸偏移量 y軸偏移量 模糊半徑 擴散半徑 顏色 內外陰影

### 盒子-正圓形狀

給正方形盒子設置圓角屬性值為 寬高的一半/50%

### 盒子-膠囊形狀

給長方形盒子設置圓角屬性值為 盒子高度的一半

### 外邊距合併-合併現象

垂直的兄弟元素 上下margin會合併 margin值較大的會生效

### 外邊距合併-塌陷

父子級標籤 子級添加上外邊距(margin-top) 會讓父級一起向下移動

解決方法: 

**1.取消給子及增加上外邊距(margin-top)，改為給父級增加內邊距(padding)**

2.父級設置 overflow:hiden(溢出隱藏)

3.父級設置 border-top(給邊框線)

### 行內元素-內外邊距問題

行內元素添加margin和padding 無法改變元素垂直位置(左右位置可生效)

解決方法:

**給行內元素增加line-height**
