# 文字屬性

### 文字大小:font-size

### 文字粗細:font-weight

 常用: 正常文字400  加粗700

### 文字傾斜:font-style

常用:清除文字預設的傾斜效果 normal

### 行高(文字垂直間距):line-height

常用:數字+px/數字(當前fontsize屬性值的倍數)ex.字體16px 2的話行高變36

 

### 字體族:font-family

font-family: Arial,'思源黑體',sans-serif;

網頁會從第一個字體開始查找,沒有的話跳到下一個,都沒有的話會跳到最後的sams-serif(無襯線字體)

### 文字複合屬性:font

一個屬性控制多個值的寫法,須按順序書寫,中間用空白隔開

font:是否傾斜 字體粗細 字體大小/行高 字體;

font:italic 700 30px/2 標楷體;

### 文字縮進:text-indent

單位

數字+px

數字+em(推薦1-2em=當前標籤的字體大小ex :16px 1em=16px 2em=32px)

### 文字對齊:text-align

text-align: left/center/right

本質是控制內容(文字)的對齊方式

可以用於控制圖片屬性置中 但需用塊級元素包裹 因為text-align只能作用於塊級元素

### 文字修飾線:text-decoration

none:無 underline:下滑線 line-through:刪除線 overline:上划線

### 文字顏色 color

color:rgb(R,G,B)

color:rgba(R,G,B,A)帶透明度文字
