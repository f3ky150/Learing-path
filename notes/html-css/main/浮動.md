# 浮動

## 浮動

讓元素水平排列 浮動的box會脫離標準流的框架

屬性名:float

屬性值

left:左對齊

right:右對齊

### 浮動特性

1. 浮動後的盒子頂對齊
2. 浮動後的盒子具備行內塊元素特點
3. 父級寬度不夠，浮動的子級會換行
4. 浮動後的盒子會脫離標準流，所以要清除浮動影響

### 清除浮動

浮動因為不在標準流之內 所以如果父級沒有高度的標籤(盒子高度) 子級添加浮動時

無法撐開父級的盒子(標準流之下 子級的盒子 會撐開父級的盒子) 

所以下一行的塊級元素 會直接疊在父級的塊及元素上 造成錯誤

解決方法

方法一:額外標籤法

在父元素裡面添加一個塊級元素，設置css屬性 clear:both

方法二:單偽元素法

```css
  父級類別::after {
	content: "";
	display: block;
	clear: both;
	}
```

方法三:雙偽元素法

```css
.父級類別::brfore,   /*解決外邊距塌陷*/
.父級類別::after {
	content: "";
	display: table;
	}
.父級類別::after {
	clear: both;
	}
```
